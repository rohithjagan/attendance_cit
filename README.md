# 📚 Attendance CIT

A modern attendance management system built for educational institutions to streamline student attendance tracking, management, and reporting.

---

## ✨ Features

- 📌 Student attendance tracking
- 🧑‍🏫 Faculty/admin management
- 📊 Attendance reports and analytics
- 🔐 Authentication & authorization
- 📱 Responsive user interface
- ⚡ Fast and lightweight architecture
- ☁️ Database integration

---

### Frontend

- CSS3
- JavaScript

### Backend
- Node.js

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/rohithjagan/attendance_cit.git
cd attendance_cit
```

## 2️⃣ Install Dependencies

```bash
npm install
```

## 3️⃣ Configure Environment Variables

Create a `.env` file in the root directory and add the required values.

### Example

```env id="2r0k61"
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

## 4️⃣ Run the Application

```bash
npm start
```
